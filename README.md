# WorkSpace Planner — PWA

Your personal schedule, notepad & resource planner as a mobile app.
Works on Android & iPhone. Free to deploy. Real push notifications.

---

## 🚀 Deploy in 5 Minutes (FREE)

### Step 1 — Get the code on GitHub

1. Go to https://github.com and create a free account
2. Click **"New repository"** → name it `workspace-planner` → click **Create**
3. Upload all these files to the repository (drag & drop the folder)

### Step 2 — Deploy on Vercel (Free)

1. Go to https://vercel.com and sign up with your GitHub account
2. Click **"Add New Project"**
3. Select your `workspace-planner` repository
4. Set these build settings:
   - **Framework Preset:** Create React App
   - **Build Command:** `npm run build`
   - **Output Directory:** `build`
5. Click **Deploy** — wait ~2 minutes
6. You get a free URL like: `https://workspace-planner.vercel.app`

### Step 3 — Install on Android Phone

1. Open Chrome on your Android phone
2. Go to your Vercel URL
3. Tap the **"Add to Home Screen"** banner that appears (or tap ⋮ menu → "Add to Home screen")
4. Tap **Add** — the app icon appears on your home screen!
5. Open the app → tap 🔔 to **enable notifications**

### Step 4 — Install on iPhone (iOS)

1. Open **Safari** on your iPhone (must be Safari, not Chrome)
2. Go to your Vercel URL
3. Tap the **Share button** (box with arrow at bottom)
4. Scroll down → tap **"Add to Home Screen"**
5. Tap **Add** — app icon appears!

> ⚠️ Note for iPhone: iOS has limited push notification support for PWAs.
> Notifications work in Safari on iOS 16.4+. For older iOS, in-app toasts still work.

## 📱 Features

- **Dashboard** — Today's brief, tomorrow preview, week strip, upcoming alerts
- **Schedule** — Tasks with type (Test, Interview, Meeting, etc.), priority, reminders
- **Notepad** — Full notepad with pinned notes, color themes, search
- **Resources** — Save AI tools, websites, links with usage descriptions & tags
- **Push Notifications** — 15 min before & at task time (Android + iOS 16.4+)
- **Offline Support** — Works without internet after first load
- **localStorage** — All data saved permanently in your browser

